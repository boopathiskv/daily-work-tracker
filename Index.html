⚠ Important — Read Before Use

  • In-memory data only. Closing or refreshing the page will clear all tasks.
  • Export JSON to save your work, and Import JSON to reload it next time.
  • Task IDs are auto-generated from the Start Date in the format YYYYMMDD## (max 99 per date).
  • Overdue tasks are highlighted when Due Date is before today and Status is not Completed.
  • No tracking, no analytics, no telemetry. The app runs entirely on your device.
  • You are solely responsible for your data. Always back up your JSON exports.

🛡 Security

  • Strict Content Security Policy (CSP) via meta tag
  • No eval(), no Function(), no inline scripts
  • All user input is sanitized and rendered via textContent
  • No innerHTML usage with user-supplied data
  • X-Content-Type-Options, X-Frame-Options, Referrer-Policy headers
  • Duplicate Task ID prevention & safe JSON import via try/catch
  • Confirmation prompts for all destructive actions
  • File-size limit (max 5MB) on JSON import
